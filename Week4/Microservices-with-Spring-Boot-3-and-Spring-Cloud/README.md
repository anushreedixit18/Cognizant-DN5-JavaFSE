# Week 4 – Microservices with Spring Boot 3 and Spring Cloud

This repository contains the Week 4 mandatory hands-on exercises from the Cognizant Digital Nurture 5.0 Java FSE program.

## Exercises

### Exercise 1 – Creating Microservices for Account and Loan
- Created Account and Loan microservices
- Implemented REST APIs using Spring Boot
- Built projects using Maven
- Tested services successfully

### Exercise 2 – Create Eureka Discovery Server and Register Microservices
- Created Eureka Discovery Server
- Registered microservices with Eureka
- Developed a Greet Service
- Implemented Spring Cloud API Gateway
- Configured service discovery using Eureka Client
- Implemented a Global Filter to log incoming requests
- Verified communication through the API Gateway

## Technologies Used

- Java 17
- Spring Boot 3
- Spring Cloud
- Spring Cloud Eureka
- Spring Cloud Gateway
- Maven
- REST APIs
