# Exercise 1 - Configuring a Basic Spring Application

## Objective

Create a basic Spring application using the Spring IoC Container and XML-based configuration.

## Description

This exercise demonstrates how to configure and manage Spring beans using the `applicationContext.xml` file. The application initializes `BookRepository` and `BookService` beans and retrieves them using the Spring IoC Container.

## Technologies Used

- Java
- Spring Core
- Spring IoC
- Maven

## Files Included

- BookRepository.java
- BookService.java
- LibraryManagementApplication.java
- applicationContext.xml
- pom.xml

## Project Structure

```text
Exercise1-ConfiguringBasicSpringApplication
│
├── BookRepository.java
├── BookService.java
├── LibraryManagementApplication.java
├── applicationContext.xml
├── pom.xml
└── README.md
```

## Learning Outcome

- Understand Spring IoC (Inversion of Control)
- Configure Spring beans using XML
- Load beans using `ApplicationContext`
- Build a basic Spring application with Maven
