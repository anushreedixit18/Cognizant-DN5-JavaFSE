# Exercise 4 - Creating and Configuring a Maven Project

## Objective

Create and configure a Maven project named LibraryManagement and add the required Spring dependencies.

## Project Details

- Project Name: LibraryManagement
- Build Tool: Maven
- Java Version: 1.8

## Dependencies Added

- Spring Context
- Spring AOP
- Spring WebMVC

## Maven Plugin

The Maven Compiler Plugin is configured to compile the project using Java 1.8.

## Files

- pom.xml
- Read.me

## Learning Outcome

This exercise demonstrates how to create a Maven project, manage Spring dependencies, and configure the Java compiler version using the Maven Compiler Plugin.
